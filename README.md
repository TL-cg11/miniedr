\# Mini EDR



A lightweight Endpoint Detection and Response (EDR) agent for Windows, written in C++.



\## Project Overview



This project is being developed in three phases:



1\. \*\*Phase 1\*\*: YARA-based static file scanner + real-time file system monitoring

2\. \*\*Phase 2\*\*: Behavior-based ransomware detection (process monitoring, heuristics)

3\. \*\*Phase 3\*\*: Centralized management with Python FastAPI server + React dashboard



\## Current Status



🚧 \*\*Phase 1 — In Development\*\* 🚧



\## Tech Stack



\- \*\*Language\*\*: C++17

\- \*\*Compiler\*\*: MSVC (Visual Studio 2022)

\- \*\*Build System\*\*: CMake + vcpkg

\- \*\*Key Libraries\*\*:

&#x20; - libyara — YARA rule engine

&#x20; - spdlog — Logging

&#x20; - nlohmann/json — JSON serialization

&#x20; - SQLite — Local event storage



\## Build Environment



\- Windows 10/11

\- Visual Studio 2022 (MSVC v143)

\- CMake 3.20+

\- vcpkg (manifest mode)
